## Base URL
All request should use the following base url:
```text
http://localhost:8000
```

## Authentication

All API endpoints are authenticated using Bearer tokens. Generate an API key by clicking "Create API Key" at the top right of the dashboard screen. Set the following header whenever making request

```json
  Authorization: Bearer {TARSIER_API_KEY}
```
