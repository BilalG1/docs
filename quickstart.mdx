### Open Dashboard

Navigate to [Tarsier Dashboard](http://tarsier.vercel.app/dash)

Create an account if you do not already have one

### Create API Key

Click `Create API Key` located at the top right of the dashboard page once logged in

Make sure to save your key in a secure location

### Make a request

Make sure to replace TARSIER_API_KEY with the key generated above and URL with any valid url

```curl
curl -X POST https://tarsier.bazzled.com/page-text \
-H "Content-Type: application/json" \
-H "Authorization: Bearer TARSIER_API_KEY" \
-d '{
  "url": "URL"
}'
```
