### Open Dashboard

Navigate to [Tarsier Dashboard](http://localhost:3000/dash)

Create an account if you do not already have one

### Create API Key

Click `Create API Key` located at the top right of the dashboard page once logged in

Make sure to save your key in a secure location

### Make a request

Make sure to replace YOUR_API_KEY with the key generated above and YOUR_URL with any valid url

```curl
curl -X POST http://localhost:8000/page-text \
-H "Content-Type: application/json" \
-H "Authorization: Bearer {YOUR_API_KEY}" \
-d '{
  "url": YOUR_URL
}'
```
